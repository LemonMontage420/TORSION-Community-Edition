# TORSION Community Edition

## A Custom Vehicle Physics Implementation Built Using Unity

 TORSION CE is a custom vehicle physics implementation built using Unity 2020 LTS (2020.3.48f1) designed to teach the user how to develop their own real-time vehicle physics implementations within their own environments, 
 whether it be Unity, another game engine, or an environment like MATLAB.

 * Straight up full custom vehicle physics for Unity that's easily scalable!
 * Understand the fundamental vehicle dynamics concepts that go into developing a real-time simulation
 * Learn how to build TORSION from scratch with the accompanying YouTube tutorial series: https://www.youtube.com/playlist?list=PL2uvZKBCoAYnfkfYm47nP5S6UeBPJPbEF
 * Join the Vehicle Physics Discord server and get access to VD papers, Git Repos, and a community of hundreds dedicated to vehicle physics development! https://discord.gg/zbZY6YRZqC
